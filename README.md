# Sitio estático LBC - listo para GitHub + Netlify

Este repositorio contiene la versión estática del sitio de LBC Consultores Ambientales.

## Estructura

- `index.html`
- `servicio.html`
- `proyectos.html`
- `proyecto.html`
- `publicaciones.html`
- `privacidad.html`
- `assets/`
  - `site.css`
  - `site.js`
  - imágenes, video y documentos

## Publicación

- **GitHub Pages:** configura la rama `main` y la carpeta raíz `/`.
- **Netlify:** no requiere build; solo despliega el sitio estático.

## Notas

- El sitio usa HTML, CSS y JavaScript sin dependencias de compilación.
- Los textos y contenidos están preparados para funcionar en UTF-8.
